⚙️ Installation

Guide pas-à-pas pour préparer l'environnement et installer Alexa Advanced Control

Prérequis

  • Python 3.11+ (venv recommandé)
  • pip
  • Node.js (optionnel pour alexa_auth/nodejs)

Note

Les commandes d'installation détaillées se trouvent dans la section Configuration ci-dessous.

📦 Étapes d'installation

# Créer et activer un venv (Windows PowerShell)
    python -m venv .venv
    .\.venv\Scripts\Activate.ps1

    # Installer les dépendances
    pip install -r requirements.txt

    # (Optionnel) Installer les dépendances de développement
    pip install -r requirements-dev.txt

🔧 Node.js (optionnel)

Si vous utilisez la récupération automatique de cookies, installez Node.js et exécutez :

cd alexa_auth/nodejs
    npm install

✅ Vérifications post-install

# Vérifier la CLI
    python alexa -h

    # Tester l'auth
    python alexa auth status

⚙️ Configuration

# Vérifier le chemin du python dans le venv
    python -c "import sys; print(sys.executable); print(sys.prefix == sys.base_prefix)"

🔎 Dépannage

Vérifiez les permissions, l'activation du venv et les logs d'erreur en cas de problème.
